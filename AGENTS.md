# AGENTS.md

このドキュメントは、本ディレクトリの改修を行う際のガイドラインです。

- ユーザーには日本語で対応してください。
- 修正を行なった後は必ずテストが成功することを確認してください。
- 変更内容をドキュメントに反映させてください。
  - MCPサーバーの利用者向けの情報はREADME.mdに記載してください。
  - 開発者向けの情報はCONTRIBUTING.mdに記載してください
- Issueの対応を行なった場合は、変更内容をIssueに反映してください。
- ライブラリの追加・削除を行なった場合は、`licensecheck`コマンドを使ってライセンスを確認し、`THIRD_PARTY_LICENSES`ファイルを更新してください。
- ユーザーからの指示の中で、今度も同様に対応が必要なものがあれば、このドキュメントに追記してください。
